# Validate Ideas

一个 AI 驱动的交互式咨询师 Skill，通过结构化的多阶段评估流程，帮助你验证一个想法是否值得投入。

## 功能简介

本 Skill 将 AI 转变为产品咨询师，引导你完成 7 个阶段的评估：

1. **想法澄清** — 将模糊的想法梳理为清晰的产品概念
2. **市场调研** — 搜索分析竞品与市场格局
3. **需求验证** — 通过社区讨论和趋势数据验证真实用户需求
4. **技术可行性评估** — 评估开发难度和资源需求
5. **商业模式分析** — 设计可行的盈利和增长策略
6. **风险评估** — 识别潜在风险并制定应对方案
7. **综合评分与最终建议** — 结构化评分及可执行的行动计划

## 核心特性

- **交互式咨询，非一次性报告** — 分阶段引导，逐步追问关键细节
- **实时调研** — 主动搜索竞品、市场数据和社区讨论
- **Idea 扩展与延伸** — 当想法存在根本性障碍时，提炼底层模式并推荐更有潜力的替代方向
- **不做简单否定** — 始终基于调研数据提供替代方案

## 使用方式

触发短语：

- "帮我验证这个想法"
- "这个 idea 值得做吗"
- "帮我做市场调研"
- "分析一下这个产品的需求"
- "validate my idea"
